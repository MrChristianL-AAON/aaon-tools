# AAON Tools

## Purpose
The following project houses the *AAON-Tools* internal application as a means for engineers and internal AAON users to interact with the Stratus unit manager Updater and Command projects

### Stack
1) Svelte 5
2) Tailwind 4
3) FastAPI
4) Rust
5) Python
6) Docker
7) Bash
8) Yocto Linux
